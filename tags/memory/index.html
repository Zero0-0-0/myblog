    Expression

    我能看到我文字的笨拙和幼稚，又在一些地方太刻意和用力过度。但是我也清楚地明白我最大的问题就是只理解，只看而不去创造。所以我能做的只有把它们狼狈地写出来。这些文字只要我没有特别说明，就没有任何ai的东西。我会问一些知识性的东西，或者把写的东西发给ai看，让它们点评，但是不会按照它们的建议修改。我希望尊重自己本来的文字，接受它们的笨拙和不成熟，然后让下一篇更好。 我们在说话的时候，总是能够精准地捕捉到自己的难堪，其最大的作用就是让自己感到更难堪。《本杰明•巴顿奇事》里提到老人会不断地重复讲述相同的观点和故事，我也会这样，并且非常严重。我的状态是社交极少但是自己感觉不到，和人相处的时候很多是一直很淡的熟人吃个饭，出去逛一下什么的。如果对方是地面，那我就是一个弹力球，弹一下，飞一会儿，过了很长时间再弹一下。很多话都是在脑海中盘旋，最后还是决定不说。我其实可以像对待一个认识了很久了的人一样对待一个比较陌生的人，但是我想这样很容易被认为是奇怪或没有边界感。这样一来能聊的话题就很有限，我一直在重复那些内容。遗憾的是，重复的内容并不一定是最重要或最能代表一个人的，一些可能根本不重要甚至有害的内容在这样的重复中得到了错误的加强。久而久之，不管是对于对方还是我自己，我会慢慢变成我故事里说的那个人。而那些故事里的我是服务于这些对话的，真正本质的东西却被遗忘了。 其实身边的人对我影响非常大，即使只有一面之缘也是，我会不由自主地想起他们，行动里也带上他们的影子。我不知道是不是他们也这么认为。我相信是这样的时候，感觉到似乎不是；我相信不是这样的时候，又感觉这是一个陷阱。既然我没有这么告诉别人，那别人也没有理由这么告诉我。唉，所以有的时候真羡慕三体人啊。

    Friend

    很奇怪，那些属性离我最远的人，反而被我反复想起。因为在不断扩展自己的领域、学习新的东西，过去的对话便被拿过来反复回味。我终于明白当时你为什么爱它们，感叹你的先进，认识到自己的局限性……唉，和自己不一样的人待在一起是很快乐的。 你千万别跟任何人谈起任何事情。你只要一谈起，就会想念起每一个人来。 ——《麦田里的守望者》 是的，连和自己都不行。
